Serendipity Project - Demo Q&A (Current)

Quick answers and code references for the final Serendipity demo.

1) Data Loading (prog.out)

  • How is prog.out parsed? Marker/value pairs (1–9) per record; trims, converts qty/wholesale/retail, and inserts into the ordered list.
  • File: mainmenu.cpp (loadData).

2) Ordered Linked List

  • What structure? Custom OrderedLinkedList<bookType*> storing pointers.
  • How is order chosen? bookType::sortCode drives comparison operators; insert keeps order.
  • Files: orderedLinkedList.h (nodes, insert, compare), booktype.cpp (operators and sortCode).

3) Lookup Variants

  • Can you return hop count or node pointer? Yes: lookUpBookHopCount (hops from head) and lookUpBookNodePtr; UI still uses bookType*.
  • Files: invmenu.cpp (helpers), orderedLinkedList.h (headPtr()), invmenu.h.

4) Memory & Ownership

  • Who owns books? Inventory owns bookType*; main deletes each book pointer on exit; list destructor frees nodes.
  • Files: mainmenu.cpp (cleanup loop), orderedLinkedList.h (destroyList).

5) Cashier Module

  • Multi-book cart? Dynamic int arrays for indices and quantities; resized as needed.
  • Oversell prevention? Tracks qty already in cart per title; caps to remaining stock; blocks if cart holds all available copies.
  • Date handling? Uses menu::currentDate() automatically—no prompt.
  • UX indicator? Bordered header “Cashier Module • Active Checkout” with date and cart count; thank-you message on success.
  • Files: cashier.cpp; date helper in menuutils.cpp/h.

6) Reports

  • Menu styling? Bordered, highlighted menu like other modules.
  • Pagination? repListing (20/pg), repWholesale (10/pg), repRetail (15/pg); arrow/any-key advance with rubber-band wrap.
  • Headers & totals? Current date in header; wholesale/retail totals shown on last page.
  • Files: reports.cpp (menu renderer, reports, pagination), menuutils.cpp/h (currentDate).

7) Inventory CRUD

  • Add/edit/delete/search? Duplicate-ISBN checks; edits are in place; deletes delete the book then remove the node; search is case-insensitive on title/ISBN.
  • Files: invmenu.cpp, bookinfo.cpp.
